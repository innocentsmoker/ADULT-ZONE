# PRIVATE ZONE - Premium Subscription Content Platform

A full-stack premium subscription-based content platform with creator subscriptions, pay-per-view content, and M-Pesa payment integration.

## 🚀 Features

### 👤 User Features
- User registration with age verification (18+)
- JWT authentication
- Browse and discover creators
- Subscribe to creators (monthly subscription)
- Pay-per-view content unlocking
- Wallet system with M-Pesa deposits
- Transaction history

### 🎨 Creator Features
- Creator account registration
- Upload images and videos
- Set subscription prices
- Dashboard with earnings analytics
- Content management
- View subscriber statistics

### 👑 Admin Features
- Admin dashboard with analytics
- User management (activate/deactivate)
- Content moderation (approve/reject)
- Report and flag viewing system

### 💳 Payment Integration
- M-Pesa payment gateway simulation
- Wallet deposits
- Automated transaction recording

## 🛠️ Tech Stack

### Backend
- Node.js with Express
- MongoDB with Mongoose
- JWT Authentication
- Multer for file uploads
- bcryptjs for password hashing

### Frontend
- React.js
- Axios for API calls
- Context API for state management
- Custom CSS with dark theme

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB installed locally or MongoDB Atlas account
- Git
